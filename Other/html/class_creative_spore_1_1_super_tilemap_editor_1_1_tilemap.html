float CreativeSpore.SuperTilemapEditor.Tilemap.InnerPadding = 0.1f

The size, in pixels, the tile UV will be stretched. Use this to fix pixel precision artifacts when tiles have no padding border in the atlas.

const int CreativeSpore.SuperTilemapEditor.Tilemap.k_chunkSize = 60

In the worst case scenario, each tile subdivided in 4 sub tiles, the maximum value should be <= 62. The mesh vertices have a limit of 65535 62(width)*62(height)*4(subtiles)*4(vertices) = 61504 Warning: changing this value will break the tilemaps made so far. Change this before creating them.
